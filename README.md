# Landing Page — Empresa Exemplo

Projeto estático feito com HTML, CSS e JavaScript puro.

## Incluído

- Layout responsivo para desktop e celular
- Animações CSS
- Animações ativadas durante a rolagem
- Botões clicáveis de telefone, WhatsApp e e-mail
- Botão flutuante de WhatsApp
- Barra fixa de contato no celular
- Preservação de parâmetros UTM no WhatsApp
- Eventos para Google Analytics, GTM e Meta Pixel
- Política de Privacidade
- SEO básico

## Arquivos

- `index.html`
- `politica-de-privacidade.html`
- `assets/css/style.css`
- `assets/js/main.js`
- `assets/img/favicon.svg`

## Substituições obrigatórias

Pesquise e substitua:

- `Exemplo`
- `(34) 0000-0000`
- `+553400000000`
- `(34) 90000-0000`
- `5534900000000`
- `contato@exemplo.com.br`
- horários de funcionamento
- texto da seção Sobre nós

O número usado no link do WhatsApp deve ficar no formato internacional,
sem espaços, traços ou parênteses.

Exemplo:

`https://wa.me/5534999999999`

## Executar localmente

Abra `index.html` no navegador ou use a extensão Live Server no VS Code.

## Rastreamento

Cole os scripts oficiais do Google Tag Manager, Google Analytics e Meta Pixel
na área comentada dentro do `<head>` do arquivo `index.html`.

Os cliques já disparam o evento `contact_click`.
